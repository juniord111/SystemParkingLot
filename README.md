# Sistema de Parqueadero en Java

Proyecto en Java utilizando programación orientada a objetos y Swing.

## Funcionalidades
- Registro de entrada y salida de vehículos
- Cálculo automático de tarifas por tiempo
- Interfaz gráfica con Swing
- Uso de enums para tipos de vehículo

## Tecnologías
- Java
- Swing
- POO
